# 阅卷辅助器 BetterGrader · 文档

> 一个**单人、本地优先**的 Tauri v2 桌面程序，用来批改**奥赛纸质大题**。
> 价值锚只有一个：**键盘驱动的判分循环**——把翻页寻址、手眼往返录入、事后聚合这些非判断性损耗全部消灭，只留「看题—给分」一个动作，并让它在键盘上快到极致。

## 一分钟上手

1. **建考试**：「考试设置」→ 新建考试…（选一个空目录，会生成 `exam.db` + `images/`）。
2. **搭骨架**：建题 + 每题满分 + 判分档位（数字键 1–9）+ 花名册（缺考清单也靠它）；可给每题填 Markdown **评分标准**。
3. **进卷**：
   - **PDF 文件夹**（每个 PDF = 一个学生，首页=姓名页）→ 导入即自动标注，可直接开批；
   - 或**图片文件夹** → 到「标注」逐页指认到 学生×题号。
4. **判分**：「判分」选题，全键盘横向批一整道题（`1–9` 给分、`Enter/→` 下一份、`R` 看评分标准…）。
5. **导出**：「导出」→ 计算汇总 → 存 CSV（Excel 可读）/ 打印报表。

## 常用命令（开发）

- Rust 领域逻辑测试（最常用）：`cargo test -p grading-core`
- 前端纯 reducer 单测：`npx vitest run`
- 运行：`npm run tauri dev`
- ⚠️ **不要**自己跑 `npm run build` / `vite build`——本环境下它会卡死（bundle 完成但 node 进程不退、堆积僵尸），类型/打包验证请手动执行或用 `npx vue-tsc --noEmit`。详见 [`CLAUDE.md`](../CLAUDE.md)。
