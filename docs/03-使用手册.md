# 03 · 使用手册

面向使用者。按流水线顺序走一遍，重点是**判分页快捷键**（命门）。

## 完整流程

### ① 新建 / 打开考试（考试设置）

顶栏「考试设置」：
- **新建考试…**：选一个**空目录**。程序在里面生成 `exam.db` + `images/`。这个目录就是这场考试的全部——可整体拷贝、归档、交接。
- **打开考试…**：选一个已有考试目录。
- **新建演示考试…**：灌一套假数据（3 题 5 人），用来体验判分手感。

> 一个目录只装一场考试。目录里已有考试时，「新建」会被拒。

### ② 搭考试骨架（考试设置）

打开考试后：
- **题目**：「＋ 添加一题」逐题加（变长列表，每题**满分各自填**）。每题可自定义**判分档位**——绑数字键 **1–9**，如「前两问=9」。建题时自动预置三档：`满分`(键 7) / `零分`(键 8) / `空白`(键 9)。改满分时「满分」档位会跟着变。
- **评分标准**（可选）：每题下面一个文本框，填 **Markdown** 要点/参考答案。判分时按 `R` 呼出对照，左侧面板也常驻一段截断预览。
- **花名册**：录学生（姓名 + 考号）。它既是标注时的选人来源，也是**到场清单**（谁没交卷 = 缺考，全靠它）。支持改名/删除。

> 判分前**必须先建题**——PDF/图片导入**不会**自动建题。

### ③ 进卷：两条导入路径

都在「考试设置」，打开考试后可见：

- **导入 PDF 文件夹…**（推荐，一键）：选一个装满 PDF 的目录，**每个 PDF = 一个学生**，文件名去掉 `.pdf` 就是姓名。约定**第 0 页 = 姓名页**，之后第 k 页 = 第 k 题。导入即**自动标注**，可直接去「判分」。渲染坏的 PDF 会先失败、不留空学生；失败的会列出来可重试。
- **导入图片文件夹…**：选一个装散图的目录（jpg/jpeg/png/webp），程序按**自然排序**（`2 < 10`）拷进 `images/` 并建**未标注**页。之后要去「标注」逐页指认。

> ⚠️ 别对同一场重复导入图片文件夹——会重复建页。

### ④ 标注（仅图片导入需要，PDF 已自动标注）

顶栏「标注」，逐页把扫描页指认到 **学生 × 题号**：
- 翻到某学生的**姓名页**，按 `S` 从花名册选人 → 这页记为姓名页（题 0），之后开始派题。
- 之后每页按 `Enter` 把「当前题号」派给这个学生、题号 +1、前进。一题多页用 `C`（接上一题、题号不变）；某题没作答页用 `N`（跳题）。
- **只切题**：侧栏有输入框可直接改**当前页**的题号（保留已绑的学生）；翻页时它会同步成本页真实题号。
- **确认总表（§7.0）**：底部按每个学生列出「答题页数 / 题数 / 是否相符」。**页数 ≠ 题数**的行会标红——要么逐页指认，要么用该行的「题号 +1 / −1」按钮**整叠平移**（治「整摞题号错一位」，比如没有姓名页导致全体偏移）。缺考学生（名册有、无卷）单独列出。

### ⑤ 判分（命门）

顶栏「判分」：上方选题 → 全键盘横向批完这一整道题 → 换下一题。详见下面的快捷键。

### ⑥ 导出

顶栏「导出」：
- **计算成绩汇总**：先看**覆盖率**——花名册 N 人、缺考几人、已判/存疑/未判各几。
- **保存 CSV…**：真文件，带 UTF-8 BOM（Excel 直接认中文）。可勾**含评语列**（所有分列在前、所有评语列在后）。
- **打印 / 导出 PDF**：走系统打印（可另存 PDF）。
- **排名门槛**：只有**所有非缺考学生每题都判完**才出排名列；还有未判/存疑就不出（不用带洞数据排假名次）。未判/存疑在表里**留空或标 `?`**，缺考整行标「缺考」——**绝不静默填 0**。

## 判分页快捷键

前置：评语框获焦时打字不会被判分键拦截；`Ctrl/Meta/Alt` 组合键放行给系统。

### 给分

| 键 | 行为 | 前进? |
|---|---|---|
| `1`–`9` | 按对应**档位**给分（写库=该档分值 + 已判） | 否（停原地复核） |
| `M` / `0` | 进入**手动输入**模式 | 否 |
| 手动态 `0-9` | 输入数字 | 否 |
| 手动态 `Enter` | 确认数字落分、退出手动、**留原地** | 否 |
| 手动态 `Backspace` | 删末位数字 | 否 |
| 手动态 `Esc` | 取消手动、不落分 | 否 |

> 手动给分要按**两次** `Enter`：第一次确认数字（留原地给你复核），想前进再按一次。能用档位就别手动——手动丢掉了「卡在哪一步」的诊断信号。

### 换人 / 翻页 / 跳转

| 键 | 行为 |
|---|---|
| `→` 或 `Enter` 或 `Space` | **下一份**（换人） |
| `←` 或 `Backspace` | **上一份**（换人） |
| `↑` | 看该生**上一页**（只读速览） |
| `↓` | 看该生**下一页**（只读速览） |
| `Esc` | 复位速览回本题页（**若 R/? 浮层开着，则先关浮层**） |
| `F` | **存疑**（保留当前分，状态转存疑） |
| `J` | 跳到**下一个存疑**单元 |
| `G` | 打开**队列总览** modal（点条目随机跳转；打开后除 Esc 外抑制判分键） |
| 顶栏「下一个未判 →」 | 循环跳到下一个未判 |

> 速览（`↑/↓`）是**纯只读**：右侧评分面板始终锚定当前 (学生,题号)，绝不会误把分落到你正瞄着的那页上。判分时发现标注错了，回「标注」阶段改，这里改不了。

### 参考 / 面板

| 键 | 行为 |
|---|---|
| `R` | 开/关**评分标准**全文浮层（当前题的 Markdown） |
| `?` | 开/关**快捷键**浮层 |
| `Tab` | 收起/展开左侧分布面板 |
| 滚轮 | 卷面缩放 |
| 拖拽 | 卷面平移 |
| 双击 | 卷面复位 |

### 判分页界面

- **顶栏**：题目切换 · 当前份是谁（**姓名 / 考号 / 第 i / N 份 / 三态**，自用软件不遮身份）· **分段进度条**（已判蓝 + 存疑橙）· 「下一个未判 →」。
- **中间**：卷面（缩放/平移/双击复位）。
- **右侧极简面板**：**记分卡置顶**（当前分放大 + 三态色块）→ 档位键列表 → 评语框；标题旁 `?` 按钮。
- **左侧可折叠悬浮面板**（`Tab` 切）：**本题平均分** + **判分分布直方图**（实时，校准「标准漂移」）+ **评分标准截断预览**（`R` 看全文）。

## 标注页快捷键

| 键 | 行为 |
|---|---|
| `←` / `→` | 翻页（不派标） |
| `S` | 打开花名册**选人**（当前页记为姓名页题 0，之后开始派题） |
| `Enter` / `Space` | 把「当前题号」派给当前学生，题号 +1，前进 |
| `C` | **接上一题号**（题号不变，处理一题多页） |
| `N` | **跳题**（题号 +1，停在原页，声明某题无页） |

侧栏另有「只切题」输入框（直接改本页题号，保留学生）和「确认总表」的整叠 ±1 平移按钮。

## 常见情况

- **谁没交卷？** 花名册里有、但一张卷都没绑的，标注确认总表和导出里都会自动列为**缺考**，不会被当成 0 分。
- **整摞题号错一位**（比如某人 PDF 没有姓名页）：标注「确认总表」里该学生会标红（答题页数对不上题数），用该行「题号 −1」整叠平移修正。
- **一道题两页纸**：标注时第二页按 `C`（接上一题，题号不变）。
- **奥赛放弃某题**：物理上补一张空白页占位（保证每人 = 姓名页 + N 张）；若没占位，标注时用 `N` 跳题手动指认。
- **判分标准飘了**（比如不知不觉给松了）：判分时瞟左侧**实时直方图**——如果分布呈 U 形（一堆满分一堆 0、中间空），说明档位太粗或标准太松，当场校正，不用等批完才发现。
- **想对照参考答案**：判分时按 `R` 看当前题的评分标准全文；先在「考试设置」→ 该题「评分标准」里填好（Markdown）。
